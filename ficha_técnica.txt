┌─────────────────────────────────────────────────────────────────────────────┐
│ 2. ESPECIFICACIONES TÉCNICAS - HARDWARE                                     │
└─────────────────────────────────────────────────────────────────────────────┘

Procesador Central (CPU):
    • Modelo:                Rockchip RK3588S
    • Arquitectura:          Octa-Core (8 núcleos)
    • Configuración:         4x Cortex-A76 (Alto rendimiento) + 
                            4x Cortex-A55 (Eficiencia energética)
    • Frecuencia:            Hasta 2.4 GHz

Unidad de Procesamiento Neuronal (NPU):
    • Tipo:                  NPU Integrada Rockchip
    • Rendimiento:           Hasta 6 TOPS (Tera Operations Per Second)
    • Optimización:          Aceleración hardware para modelos de IA

Memoria (RAM):
    • Capacidad:             4 GB LPDDR4 (configuración estándar)
    • Opciones:              8 GB / 16 GB LPDDR4 (según configuración)
    • Velocidad:             Alta velocidad para procesamiento en tiempo real

Almacenamiento:
    • Tipo:                  eMMC integrado / Tarjeta microSD
    • Configuración:         Flexible según necesidades del cliente
    • Capacidad:             Configurable (mínimo 32 GB recomendado)

Sistema de Alimentación:
    • Entrada:               12V DC (carga de batería o alimentación directa)
    • Batería:               Integrada con sistema de gestión de energía
    • Regulador:             Sistema de regulación automática (funciona con 
                            batería o alimentación externa)
    • Autonomía:             Operación portátil sin conexión a red eléctrica

Conectividad de Red:
    • Ethernet:              Puerto Gigabit (1 Gbps)
    • Wi-Fi:                 Doble Banda (2.4 GHz / 5 GHz)
    • Modo Access Point:     Creación de red Wi-Fi local integrada
    • Protocolos:            TCP/IP, RTMP, HTTP/HTTPS, WebSocket

Interfaces de Video:
    • HDMI:                  Salida de video digital (monitoreo local)
    • MIPI CSI:              Entrada para cámaras externas
    • Procesamiento:         Hasta 4K @ 30fps (decodificación hardware)

Sistema Operativo:

Modelos de IA Pre-configurados:
    • Detección UAV:          VisDrone YOLO11n (11 clases: personas, vehículos,

    • Detección de Fuego:    Modelo especializado para incendios

Servicios Integrados:

    • Streaming:             Servidor RTMP integrado
    • Comunicación:          Protocolo en tiempo real bidireccional

┌─────────────────────────────────────────────────────────────────────────────┐
│ 4. CAPACIDADES Y FUNCIONALIDADES                                            │
└─────────────────────────────────────────────────────────────────────────────┘

Procesamiento de Video:
    • Ingesta:               Protocolo RTMP (streaming en vivo)
    • Resolución:            640x480 (configurable hasta 4K)
    • Frame Rate:            Hasta 30 FPS en tiempo real
    • Latencia:              < 100 ms (procesamiento en milisegundos)

Detección y Análisis:
    • Selección de Clases:   Interfaz web para seleccionar objetos a detectar

    • Visualización:         Bounding boxes con colores personalizables
    • Métricas en Tiempo Real:
        - FPS de procesamiento
        - Conteo de objetos por clase
        - Latencia de inferencia
        - Estado de conexión RTMP

Interfaz de Usuario:
    • Tipo:                  Aplicación Web Progresiva (PWA)
    • Acceso:                Navegador web estándar (sin instalación)
    • Características:
        - Video en tiempo real con pantalla completa
        - Panel de control intuitivo
        - Selección de modelos de IA
        - Configuración de parámetros de detección
        - Métricas y estadísticas en vivo
        - Control de apagado seguro del sistema

Conectividad y Red:
    • Hotspot Wi-Fi:         Red local independiente (sin internet requerido)
    • IP Fija:               Configuración automática
    • Acceso Remoto:         Vía navegador web desde cualquier dispositivo
    • Streaming:             Recepción de video desde drones/cámaras IP
